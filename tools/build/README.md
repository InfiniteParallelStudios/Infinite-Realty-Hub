# Build Tools

This directory contains build scripts and utilities.

## Scripts

- `build-all.sh` - Build all packages and services
- `build-packages.sh` - Build only packages
- `build-services.sh` - Build only services
- `clean.sh` - Clean all build artifacts

## Usage

```bash
# Make scripts executable
chmod +x *.sh

# Build everything
./build-all.sh

# Clean build artifacts
./clean.sh
```
