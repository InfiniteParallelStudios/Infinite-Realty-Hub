# GitHub Labels Setup

Copy and paste these labels into your GitHub repository (Settings → Labels):

## Epic Labels
- **Name:** `epic` **Color:** `#7B68EE` **Description:** Major multi-week initiatives
- **Name:** `phase-1` **Color:** `#0052CC` **Description:** Phase 1: Foundation & Setup  
- **Name:** `phase-2` **Color:** `#0052CC` **Description:** Phase 2: UI/UX Foundation
- **Name:** `phase-3` **Color:** `#0052CC` **Description:** Phase 3: Core Platform Features
- **Name:** `phase-4` **Color:** `#0052CC` **Description:** Phase 4: CRM App Development

- **Name:** `high` **Color:** `#FF6600` **Description:** High priority items
- **Name:** `medium` **Color:** `#FFCC00` **Description:** Medium priority items
- **Name:** `low` **Color:** `#00CC00` **Description:** Low priority items
